• time: The time this attribute was recorded.

  • entity_oid: The oid of the entity the attribute pertains to.

  • type: The attribute type.

  • bool_value: If the attribute is boolean, this is the boolean value.

  • int_value: If the attribute is an integer, this is the integer value. If the attribute is an enum, this is the oid of the enum in the pg_enum table (to guard against name or order changes). The int_value should be castable to the enum via a custom cast (e.g. int _value::entity_state = 'POWERED_ON').

  • long_value: If the attribute is a long, this is the long value.

  • double_value: If the attribute is a double, this is the double value.

  • string_value: If the attribute is a string, this is the string value.

  • int_arr_value: If the attribute is a list of ints, this is the value. If the attribute is a list of enums, these are the OIDs of the enums in the pg_enum table (to guard against name or order changes).

  • long_arr_value: If the attribute is a list of longs, this is the value.

  • string_arr_value: If the attribute is a list of strings, this is the value.

  • double_arr_value: If the attribute is a list of doubles, this is the value.

  • json_value: If the attribute is a JSON object, this is the value.
