• recommendation_time: The time the action was recommended.

  • action_oid: oid of the action in the system.

  • type: The type of the action.

  • severity: The severity of the action.

  • category: The category of the action.

  • target_entity_id: The oid of the target entity.

  • involved_entities: The oids of the involved entities (includes the target entity).

  • description: The text description of the action.

  • savings: Savings (or, if negative, investment) of the action.

  • attrs: Additional (often type-specific) information for the action. For example, in resizes this will contain the commodity type and from/to amounts.
